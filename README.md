# OpenTalk Security Assessment

Аудит защищённости платформы видеоконференцсвязи с открытым исходным кодом **OpenTalk** (Rust backend, Docker-контейнеризация).

> Учебный проект. Все проверки выполнялись на изолированном демонстрационном стенде (WSL + Kali Linux), без затрагивания продуктивных систем.

## Объём работ

- Анализ архитектуры и функциональных компонентов ПО (Nginx, Keycloak, Controller, LiveKit, PostgreSQL, RabbitMQ, MinIO)
- Идентификация ПО и версий компонентов: активное сетевое сканирование (Nuclei, Whatweb, Curl) + локальное выявление
- Анализ сетевых протоколов и открытых портов
- Поиск применимых 1-day уязвимостей (NVD, БДУ ФСТЭК) под выявленные версии
- Построение и классификация цепочек эксплуатации
- Практическая верификация цепочек на стенде
- Аудит компонентов: Nginx, Keycloak, Controller/API, веб-интерфейс, слой хранения

## Ключевые находки

| Компонент | Находка | Статус |
|---|---|---|
| Keycloak | DoS через некорректную обработку заголовка Authorization на эндпоинте регистрации клиента (CVE-2026-9803) | Подтверждено, воспроизведено |
| Keycloak | Повышение привилегий через token exchange с oversized JWT (CVE-2026-9704) | Теоретически применимо, на стенде не выполняются условия эксплуатации |
| Keycloak | Обход redirect_uri (CVE-2026-7504) | Не воспроизведено (wildcard-клиенты не используются) |
| Nginx | Отсутствие защитных заголовков (CSP, X-Frame-Options, HSTS) | Подтверждено |
| Веб-интерфейс | Перебор room ID (IDOR) | Невозможен — UUIDv4, 122 бита энтропии |
